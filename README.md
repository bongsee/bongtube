# bongtube

Youtube clone project

# 요구 사항

## 사용할 API

[YouTube Data API](https://developers.google.com/youtube/v3/getting-started?hl=ko)

## 구현할 기능

1. 메인 페이지
   1. 핫 트렌드 비디오를 홈페이지에 노출
   2. 키워드 검색
2. 비디오 페이지
   1. 채널 상세 내용
   2. 연관된 비디오를 사이드에 노출
